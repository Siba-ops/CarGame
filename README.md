# Car Game

## Overview

A simple Python command-line Car Game that allows the user to start and stop a virtual car using text commands.

---

## Features

- Start the car
- Stop the car
- Quit the game
- Handles invalid commands

---

## Technologies Used

- Python 3

---

## Concepts Practiced

- Loops
- Conditional Statements
- User Input
- Program Flow

---
